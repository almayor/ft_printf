char* ft_strsub ( char const *  s,
unsigned int  start,
size_t  len 
)

Allocates (with malloc) and returns a “fresh” substring from the string given as argument. The substring begins at index start and is of size len. If start and len aren’t refering to a valid substring, the behavior is undefined. If the allocation fails, the function returns NULL.

Parameters
sThe string from which create the substring.
startThe start index of the substring.
lenThe size of the substring.
Returns
The substring.
